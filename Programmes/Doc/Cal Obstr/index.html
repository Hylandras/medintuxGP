Description

Ce programme est le plugin de calcul de calendrier obstétrical de MedinTux.

il permet à partir de la date des dernières règles sous format : dd-MM-yyyyy de générer un calendrier obstétrical à insérer dans n'importe quel documment de MedinTux.

Pour appeler ce plugin et l'activer il suffit par exemple de créer une entrée ou item de menu déroulant de la forme ci-dessous :

$start_title Calendrier Obstétrical $end_title{{[Execute]
pathPlugin=../../Plug/cal_obst/bin/cal_obst
maskExch=Bonjour tout le monde
pathIni=****
waitEnd=y
[Parametres]
m_DateDerRegle = {{DDR=dd-MM-yyyyy}}
}}

le fichier :calendrier obstetrical.html contient un exemple de retour.

Lors d'un click sur l'item le calendrier obstétrical sera inséré, à l'endroit du texte où se trouve le curseur.
